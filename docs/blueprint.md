# **App Name**: FocusFlow

## Core Features:

- Task Creation: Allow users to add tasks with a title and category (Study, Work, Personal).
- Task Categorization: Enable users to categorize tasks for better organization.
- Task Completion: Enable users to mark tasks as completed.
- Task Deletion: Allow users to delete tasks.
- Local Storage: Store tasks locally in the browser using localStorage.
- Task Prioritization (AI): Suggest task prioritization based on category. Generative AI tool analyzes the task categories to determine order, for example prioritizing 'Work' tasks higher during business hours.

## Style Guidelines:

- Primary color: Soft blue (#A0C4FF) to promote calmness and focus.
- Background color: Light, desaturated blue (#EBF4FF) to maintain a clean and professional look.
- Accent color: Muted orange (#FFB3A7) for interactive elements and highlights.
- Body and headline font: 'PT Sans' for a clean, modern and easily readable experience.
- Use a clean and modern layout with good spacing to enhance readability and reduce clutter. Prioritize mobile responsiveness.
- Use simple, consistent icons to represent task categories and actions (e.g., pencil for edit, trash can for delete, checkbox for complete).
- Add subtle animations for task completion and deletion to provide visual feedback.
